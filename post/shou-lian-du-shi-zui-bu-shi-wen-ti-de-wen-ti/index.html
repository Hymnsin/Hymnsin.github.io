熟练度是最不是问题的问题

2019-05-25

最近利用墙外的自由互联网环境尝试了一圈 真·匿名上网，然后发现有很多坑绕不过去，有些苦恼。

除了速度以及身份验证等匿名上网必然会导致的问题之外，Linux 系统的使用也是一大障碍。过去的人生当中，都没有认真使用过 Linux 系统，因此非常不熟悉。Terminal 命令行只会按照教程一个字一个字敲进去，完全不会自己按需调配；甚至连 GUI 都不会用，面对被隐藏的文件夹、没有 write 权限的文件，都只能临时上网搜索解决方案；至于 network 设置、proxy 设置、什么情况下要用 sudo，可能在熟练者眼中非常基础的内容，我也半懂不懂——如此种种，不禁产生畏难心理，觉得自己恐怕是连个 Linux 系统都用不好了。

然而，事情可能没那么糟糕。我可能只是不熟悉而已。

回想上手 macOS 之前，我不也是在 Apple Store 里试用演示机的时候，连如何最小化窗口、如何打开 Dock 栏以外的程序都不知道操作吗？至于如何在 Terminal 当中敲代码，我当时是想都不敢想——然而经过 macOS 的日常使用，渐渐接触多了以后，这些本来看起来很遥远的东西，早就在潜移默化当中学会了。

所以说：熟练度是最不是问题的问题。只需要时间以及投入，总能化不熟练为熟练。
